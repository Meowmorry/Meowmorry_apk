# Meowmorry 🐱

Meowmorry adalah platform web untuk dijadikan sebagai ruang curhat yang nyaman bagi penggunanya dengan maskot kucing yang lucu.

## 👥 Tim Pengembang
- **Team Leader:** Iin Permatasari
- **UI/UX:** Dinda Radesti Listyanto
- **Front-End:** Chantikka Revina
- **Back-End:** Iin Permatasari

## 🚀 Tech Stack
- **Backend:** Node.js, Express
- **Database:** Supabase
- **Design:** Figma 

## 🛠️ Cara Memulai (Local Development)
1. Clone repositori ini.
2. Jalankan `npm install` untuk menginstall dependencies.
3. Buat file `.env` dan masukkan kredensial database.
4. Jalankan `npm start` atau `npm run dev`.
